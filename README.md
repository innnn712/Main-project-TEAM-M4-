  어노테이션 형식

  - Polygon segmentation (픽셀 단위) — 파일명의 (pixel)이 이를 의미
  - 각 어노테이션에 segmentation (폴리곤 좌표 리스트), bbox, area 포함
  - 이미지당 평균 약 11.5개의 어노테이션

  images 폴더

  600개 파일이 있으며, 이미지 1장당 3개 파일로 구성:
  - night ride (N).png — 원본 이미지
  - night ride (N).png___fuse.png — 세그멘테이션 오버레이 시각화
  - night ride (N).png___save.png — 세그멘테이션 마스크

  이 데이터셋을 활용하면 야간 주행 환경에서의 도로 영역 분할, 차선 검출, 객체 탐지 등의 딥러닝 모델을 학습할 수 있습니다
